This folder holds the active engineering documentation for Pulse.

   Core product intent and engineering decision rules.
2. `architecture-and-core-logic.md`
   Current backend and frontend architecture with runtime flow.

4. `operations-maintenance.md`

## History policy

1. Move any completed phase plan to `Doc/history`.
2. Move any superseded document to `Doc/history`.
3. Keep only current implementation guidance in this folder.
